About Our Website

Welcome to the About page! This page demonstrate how I maintain consistent structure and styling across all pages of our multi-page website.

My objective

I believe in creating web experiance that are both beautiful and funcional.My aim is to demonstrate best practices in multi-page website development, including:

  • Consistent navigation and user experiance
  • Responsive design that works on all devices
  • Clean, maintainable code structure
  • Accessibility and usability standards
  • Modern CSS techniques and animation

Technical Implementation

This website showcases several important web development concepts:

File Organization

Website is organized into logical directories:

  • CSS/-Contains our shared stylesheet
  • Root directory-Contains all HTML pages

Consistent Styling

All pages share the same CSS stylesheet(style.css), ensuring consistent appearance across the entire website. The stylesheet includes:

  • Base styles and typography
  • Navigation styling with hover effects
  • Active link highlighting
  • Smooth Animation and transitions

Navigaton System

Our navigation uses relative paths to links between pages, making the website portable and easy to deploy. The active page is highlighted using CSS class.
